# O que é o Svelte e porque utilizá-lo?

Svelte é um framework criado recentemente em 2016. Ele tem como objetivo otimizar e facilitar a criação de web páginas. Um diferencial seu é que ele pega o código escrito e o compila em JavaScript puro durante o processo de build, assim manipulando diretamente o DOM sem precisar de uma biblioteca intermediaria.
Outra vantagem é a sua curva de aprendizado, já que ele é um framework mais simples e intuitivo que o React e Angular


## Quando não utilizar

Em projetos muito grandes ou quando se necessita de muitos recursos prontos não se recomenda utilizar o Svelte já que ele não possui uma extensa biblioteca ou uma grande comunidade.  

 

## Instalação e Configuração

Para utilizar o Svelte é necessário ter instalado o node. Após a instalação do node entre na pasta do seu projeto e rode estes comandos, escolha o 
svelte como framework após escolha JavaScript ou TypeScript.  *Sempre que você quiser visualizar a página web terá que rodar o comando npm run dev na pasta src.* 

``` Para copiar
npx sv create myapp
cd myapp
npm install
npm run dev

```
